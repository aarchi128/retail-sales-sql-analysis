# Retail Sales Data Analysis using SQL

## Overview
This project performs end-to-end analysis of retail sales data using SQL to extract business insights.

## Dataset
Transactional retail sales data containing:
- Sale date & time
- Product category
- Quantity sold
- Cost and total sales
- Customer information

## Key Analysis
- Total revenue and transactions
- Average order value
- Monthly revenue trends
- Sales by day of week and hour
- Category-wise revenue and profit
- Top customers by spending
- Category ranking using window functions

## Tools Used
- MySQL
- SQL
- VS Code
- GitHub

## Files
- `schema.sql` – database and table creation
- `queries.sql` – analytical SQL queries
